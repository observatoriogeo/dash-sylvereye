## Installation

The easiest way to install Dash Sylvereye is with [pip](https://pypi.org/project/pip/):

```bash
pip install dash-sylvereye
```

You can also build Dash Sylvereye from source by cloning the [GitHub repository](https://github.com/observatoriogeo/dash-sylvereye) and running the build script. See the [build instructions](build_instructions) for details.
