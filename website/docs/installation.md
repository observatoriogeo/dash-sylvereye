## Installation

The easiest way to install Dash Sylvereye is through [pip](https://pypi.org/project/pip/):

```bash
pip install dash-sylvereye
```

You can also manually install Dash Sylvereye by getting the latest build from the [GitHub repository](https://github.com/observatoriogeo/dash-sylvereye).

Please refer to [this page](build_instructions) to find manual build instructions.
